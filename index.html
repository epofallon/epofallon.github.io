Recently, I developed Bard, an open-source session replay tool that enables the viewing and analysis of user sessions.

  • Tell Me More

Bard is an open-source session replay tool. It abstracts away the complexity of recording, analyzing, and replaying user session data, helping answer the question, "How do users experience a web application?"

Learn More

Bard puts user sessions in the context of a conversion funnel, making issues with front-end applications visible, and enabling developers to make informed improvements to an application.
